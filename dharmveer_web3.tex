\projectHeading{OP Stack Withdrawals Indexer}{https://github.com/d-Bharti001/op-withdrawals-indexer}{Golang, Optimism Stack, Blockchain, PostgreSQL}
\begin{bullets}
    \item Golang CLI app to record withdrawal initiation, proving, and finalization events related to an OP Stack chain.
    \item Two services - \textbf{Indexer} and \textbf{API} - with a shared database, for easy horizontal scalability.
    \item Interfaces are used for decoupling the application logic from blockchain and database implementations.
    \item Reliable polling based event indexing logic with retries, along with idempotent SQL queries preventing duplicate inserts, and graceful shutdown logic.
    \item \textbf{Web interface for OP Sepolia} accessible here: \weblink{https://op-sepolia-bridge.d-bharti.dev}{https://op-sepolia-bridge.d-bharti.dev}. Frontend built with \textbf{Lovable} and \textbf{Cursor}.
